## 第一部分：桌面端 QClaw 完整功能分析

### 1. 核心架构
- **Gateway**：WebSocket 服务器，监听 28789 端口

- **多通道支持**：微信、QQ、Telegram、Discord、Slack、iMessage 等

## 第二部分：手机端 QClaw 开发规划

### 架构设计

```
┌─────────────────────────────────────────┐
│         Android 手机端 QClaw            │
├─────────────────────────────────────────┤
│  React Native / Flutter UI 层           │
├─────────────────────────────────────────┤
│  Capacitor 桥接层                       │
├─────────────────────────────────────────┤
│  Node.js 运行时（Capacitor Node 插件） │
├─────────────────────────────────────────┤
│  OpenClaw Gateway（完整打包）           │
│  - Pi Agent                             │
│  - 所有 Skills                          │
│  - 所有 Plugins                         │
│  - 所有通道支持                         │
├─────────────────────────────────────────┤
│  配置管理（从桌面端导入）               │
├─────────────────────────────────────────┤
│  本地存储（SQLite/LevelDB）             │
├─────────────────────────────────────────┤
│  云端 LLM（GLM/OpenAI/Claude）          │
└─────────────────────────────────────────┘
```

### 技术栈

| 层级 | 技术 | 说明 |
|-----|------|------|
| UI | React 18 + TypeScript | 跨平台 UI 框架 |
| 样式 | Tailwind CSS v4 | 响应式设计 |
| 状态 | Zustand | 轻量级状态管理 |
| 路由 | React Router v6 | 页面导航 |
| 打包 | Capacitor 8 | Android 原生集成 |
| 后端 | Node.js + OpenClaw | 完整 Gateway |
| 数据库 | SQLite | 本地存储 |
| 构建 | Vite | 快速构建 |
